== Description ==

ACF Field to select a date with select drop downs rather than the typical popup date picker. Very useful for selecting Age/Date of Birth.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `acf-field-date-selector` folder into your `wp-content/plugins` folder
2. Activate the Advanced Custom Fields: Date Selector Field plugin via the plugins admin page
3. Create a new field via ACF and select Date Selector Field type under the jQuery section
4. Read the description above for usage instructions
